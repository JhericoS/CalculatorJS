# CalculatorJS
Simple calculator developed with Javascript

This calculator contains these operations:
- Addition (+)
- Subtraction (-)
- Multiplication (x)
- Division (/)
- Square (^)

It has the functionality to:
- Clear the operation (C)
- Remove the last character (‚Üê)
- Use negative numbers (+/-)
- Use decimal numbers (.)
